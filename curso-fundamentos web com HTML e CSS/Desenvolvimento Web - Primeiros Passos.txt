#TCP/IP, portas, roteadores, switches e modems

TCP/IP - Transmission Control Protocol/Internet Protocol
Protocolos de comunicação entre computadores em rede
4 etapas:
1- Física - ex:placa de rede
2- Rede - ex:IP
3- Transporte - ex:TCP,UDP
UDP - Rápido, livestrem e não confiável(informações podem se perder, não há complicação)
TCP - Voltado à conexão, handshake, integridade e ordem dos dados, aplicativos de mensagem
4- Aplicação - ex:FTP,SMTP,HTTP

Portas - por onde os dados saem e chegam
20: FTP - apenas envio de arquivos
22: SSH - protocolo de rede criptográfico para operação de serviços de rede de forma segura sobre uma rede insegura, uso em login remoto de utilizadores a sistema de computadores
25: SMTP - envio de email
53: DNS - converter busca em um IP
80: HTTP - requisição simples
443: HTTPS - requisição segura

Modem
Modulator-demodulator
Hardware que converte dados em um formato que possa ser transmitido de um computador e lido por outro

Rotear
Distribui internet de um ou mais dispositivos por uma rede
Pode fazer comunicação entre redes

Switch
Distribui internet para um ou mais dispositivos de uam rede
